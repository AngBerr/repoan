Синхронизация lock

Преподаватель: Белехев Алексей Олегович

Программа демонстрирует работу нескольких потоков с общим счётчиком.
Рассматриваются два варианта:

1) без использования lock;
2) с использованием lock.

Без синхронизации итоговое значение счётчика может быть неверным. Использование lock обеспечивает корректную работу потоков и правильный результат.
